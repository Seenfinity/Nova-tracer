NOVA Tracer — Agentic Engineering Grant Submission Bundle
===========================================================

This folder contains everything you need to submit the grant application
at https://superteam.fun/earn/grants/agentic-engineering

What's inside
-------------

1. GRANT_APPLICATION.md
   The complete draft of your grant form, organized exactly like the
   3-step Solana Earn form. Open this in any markdown viewer (or just
   any text editor) and copy each field into the form. Personal fields
   are marked with [ FILL ].

2. crowdedness-score.png
   Visual Crowdedness Score report — score 2/10 (Low crowding).
   Generated by Colosseum Copilot from the 5,400+ Solana hackathon
   project corpus. THIS IS THE FILE THE FORM ASKS YOU TO UPLOAD AS
   AN IMAGE LINK.

3. crowdedness-report.html
   Interactive standalone HTML version of the same report. Open it in
   any browser (no server needed) — useful if you want to take your
   own screenshot at a different size.

4. claude-session.jsonl
   Full Claude Code session transcript (~2.7 MB) — proof of
   AI-assisted development. Attach this file to the form when it asks
   for the AI session transcript.

How to submit (in order)
------------------------

1. Upload `crowdedness-score.png` to your Google Drive.
   Right-click → Share → "Anyone with the link can view".
   Copy the share link.

2. (Optional) You can upload the entire grant-submission/ folder to
   Drive instead, which makes everything publicly available in one
   place.

3. Open https://superteam.fun/earn/grants/agentic-engineering and click
   the "Apply" button.

4. Walk through the 3 form steps using GRANT_APPLICATION.md as your
   reference:
   - Step 1 (Basics): Project Title, One-liner, your TG, your wallet
   - Step 2 (Details): paste Project Details, Proof of Work, your X
     handle, your GitHub, paste the Drive link from step 1 in the
     Crowdedness Score field, attach claude-session.jsonl
   - Step 3 (Milestones): paste milestones, pick one Primary KPI

5. Submit.

For the final tranche (paid out after ship)
-------------------------------------------

When the time comes you'll need to provide:
- Colosseum project link (after submitting NOVA Tracer to Colosseum)
- GitHub repo link: https://github.com/Seenfinity/Nova-tracer
- AI subscription receipt (Claude / Anthropic invoice screenshot)
